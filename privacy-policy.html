Политика конфиденциальности

Вступает в силу: 23 мая 2026 г.  ·  Версия 1.0

Настоящая Политика конфиденциальности описывает, каким образом приложение Glumo собирает, использует и защищает данные пользователей. Мы серьёзно относимся к вашей конфиденциальности: приложение спроектировано по принципу «минимум данных».

1. Кто отвечает за данные

Приложение разработано и поддерживается:

  • Разработчик: Баюнов Арсений Романович
  • Электронная почта: baunovarseniy@gmail.com
  • Тип: индивидуальный разработчик

2. Какие данные собираются

Приложение не собирает никаких персональных данных автоматически. Единственные данные, с которыми работает Glumo:

  • Фотографии и изображения — загружаемые вами для распознавания текста. Хранятся только локально на устройстве.
  • Результаты распознавания — распознанный текст сохраняется в истории на вашем устройстве.
  • Настройки приложения — язык, тема, параметры распознавания. Хранятся локально.
Мы не собираем, не передаём и не продаём персональные данные, аналитику поведения, геолокацию или идентификаторы устройства.

3. Передача данных третьим сторонам

Передача данных происходит только при использовании дополнительных движков распознавания:

  • GlumoVision (локальная нейросеть) — изображение обрабатывается на устройстве. Никуда не передаётся.
  • GlumoVision Cloud — изображение отправляется на сервер Glumo исключительно для обработки. Не хранится после получения результата.
  • Google Gemini — изображение передаётся в Google LLC для обработки. Применяется политика конфиденциальности Google.
  • Anthropic Claude — изображение передаётся в Anthropic PBC для обработки. Применяется политика Anthropic.
  • OpenAI GPT — изображение передаётся в OpenAI для обработки. Применяется политика OpenAI.

Облачные движки доступны только на платных тарифах и используются исключительно по явному выбору пользователя.

4. Разрешения приложения

  • Камера — используется для съёмки документов и рукописных текстов. Не используется в фоне.
  • Чтение медиафайлов — используется для выбора изображений из галереи.
  • Интернет — используется только при отправке изображений в облачные движки или для отображения этой страницы.

5. Хранение и удаление данных

Все данные (история, настройки, кэш изображений) хранятся исключительно на вашем устройстве. При удалении приложения все данные удаляются вместе с ним.

Вы можете вручную очистить историю распознаваний в любой момент через раздел Настройки → Хранилище → Очистить историю.

6. Безопасность

Приложение не требует создания учётной записи. Данные не передаются на какой-либо централизованный сервер без явного действия пользователя. При использовании облачных движков соединение осуществляется по протоколу HTTPS.

7. Дети

Приложение не предназначено специально для детей до 13 лет и не собирает данные о несовершеннолетних целенаправленно.

8. Изменения политики

При внесении существенных изменений в данную политику дата версии будет обновлена. Актуальная версия всегда доступна по адресу:
