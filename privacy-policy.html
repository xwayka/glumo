2. Какие данные обрабатываются

Приложение обрабатывает только те данные, которые необходимы для его работы. Мы не требуем регистрации, не собираем имя, номер телефона, e-mail или иные идентификаторы пользователя.

3. Для каких целей обрабатываются данные

  • Распознавание рукописного текста — основная функция Приложения.
  • Исправление орфографии и перевод — при активации соответствующих опций в настройках (выполняется на сервере или через сторонние API, см. раздел 5).
  • Хранение истории распознаваний — чтобы вы могли вернуться к ранее распознанным текстам. Данные хранятся только на вашем устройстве.
  • Сохранение настроек — чтобы ваши предпочтения сохранялись между сеансами работы с Приложением.

Мы не используем ваши данные в рекламных целях и не продаём их третьим лицам.

4. Как и где хранятся данные

4.1 Данные на вашем устройстве

История распознаваний и настройки хранятся в изолированном хранилище Приложения на вашем устройстве (стандартные механизмы Android: Room Database, DataStore Preferences). Данные доступны только Приложению и не передаются другим приложениям на устройстве. При удалении Приложения все локальные данные удаляются автоматически операционной системой.

4.2 Наш сервер

При нажатии кнопки «Распознать» фотография передаётся на наш сервер по зашифрованному каналу (HTTPS). Сервер расположен на территории Российской Федерации. Изображение обрабатывается нейросетью GlumoVision (CRNN) исключительно в оперативной памяти и удаляется после формирования ответа. Сервер не ведёт логов содержимого фотографий и распознанных текстов.

8. Безопасность данных

Мы принимаем следующие меры для защиты ваших данных:

  • Передача данных между Приложением и сервером осуществляется по протоколу HTTPS (TLS 1.2+).
  • Данные на вашем устройстве защищены системными механизмами изоляции Android.
  • Ключи API сторонних сервисов хранятся на сервере в переменных окружения, недоступных пользователям и коду Приложения.
  • Фотографии и тексты не сохраняются на сервере после завершения обработки.
  • Сервер не хранит персонализированных данных, позволяющих идентифицировать конкретного пользователя.

Вместе с тем ни одна система передачи данных через Интернет не может гарантировать абсолютную защиту. Мы делаем всё возможное для обеспечения безопасности, но не можем гарантировать защиту от любых угроз.

9. Ваши права

В соответствии с Федеральным законом № 152-ФЗ вы вправе:
